CodeReview AI is a full-stack AI-powered code review application built with Node.js, Express, MongoDB, React (Vite), and Google Gemini API.
It allows users to submit code in different programming languages and receive an automated AI review including a score, detected issues, improvement suggestions, and positive feedback.

The backend handles API requests, connects to MongoDB for storing review history, and integrates with the Gemini AI model for intelligent code analysis.
The frontend provides a clean interface to submit code, view results, and manage previous reviews.
